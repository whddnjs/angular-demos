Custom Pipes Demo

31 real-world Angular custom pipe examples

#1 capitalizeFirst

첫 글자 대문자 변환

{{ 'hello WORLD' | capitalizeFirst }}

#2 titleCase

각 단어 첫 글자 대문자

{{ 'hello world from angular' | titleCase }}

#3 camelCase

camelCase로 변환

{{ 'hello world pipe' | camelCase }}

#4 reverse

문자열 뒤집기

{{ 'Angular' | reverse }}

#5 truncate

텍스트 자르기 (limit: 30)

{{ longText | truncate:30 }}

#6 truncate (completeWords)

단어 단위로 자르기

{{ longText | truncate:30:true }}

#7 excerpt

발췌문 생성 (limit: 50)

{{ longText | excerpt:50 }}

#8 slugify

URL slug 변환

{{ 'Hello World! Angular Pipes' | slugify }}

#9 replace

문자열 치환

{{ 'I love React' | replace:'React':'Angular' }}

#10 repeat

문자열 반복

{{ 'Ha' | repeat:3:' ' }}

#11 initials

이니셜 추출

{{ 'John William Doe' | initials }}

#12 mask

문자 마스킹 (카드번호)

{{ cardNumber | mask:4:'*' }}

#13 stripHtml

HTML 태그 제거

{{ htmlString | stripHtml }}

#14 safeHtml

HTML 안전하게 렌더링

#15 highlight

검색어 하이라이트

#16 defaultValue

기본값 제공

null → "{{ null | defaultValue:'No data available' }}"

#17 timeAgo

상대 시간 표시

{{ pastDate | timeAgo }}

#18 countdown

카운트다운 타이머

{{ futureDate | countdown }}

#19 fileSize

파일 크기 포맷팅

1024 → {{ 1024 | fileSize }}
1048576 → {{ 1048576 | fileSize }}
1073741824 → {{ 1073741824 | fileSize }}

#20 currencyConverter

환율 변환

$100 → {{ 100 | currencyConverter:1350.5:'KRW' }}
$100 → {{ 100 | currencyConverter:0.92:'EUR' }}

#21 percentage

퍼센트 계산

75/200 → {{ 75 | percentage:200:1 }}

#22 ordinal

서수 접미사 추가

{{ 1 | ordinal }}, {{ 2 | ordinal }}, {{ 3 | ordinal }}, {{ 4 | ordinal }}, {{ 11 | ordinal }}, {{ 21 | ordinal }}

#23 pluralize

복수형 처리

{{ 1 | pluralize:'item' }}
{{ 5 | pluralize:'item' }}
{{ 3 | pluralize:'child':'children' }}

#24 join

배열을 문자열로 결합

{{ tags() | join:' | ' }}

#25 unique

중복 제거

[1,2,2,3,3,3,4,5,5] → {{ duplicates | unique | join:', ' }}

#26 flatten

중첩 배열 평탄화

[[1,2],[3,[4,5]],[6]] → {{ nestedArray | flatten | join:', ' }}

#27 chunk

배열을 N개씩 분할

@for (group of tags() | chunk:2; track $index) {
[{{ group | join:', ' }}]
}

#28 range

숫자 범위 생성

range(1, 10, 2) → {{ 1 | range:10:2 | join:', ' }}

#29 groupBy

속성별 그룹핑

@for (entry of fruits | groupBy:'category' | keyvalue; track entry.key) {
{{ entry.key }}: @for (item of entry.value; track item.name) { {{ item.name }} }
}

#30 filter

배열 필터링

@for (fruit of fruits | filter:searchTerm():'name'; track fruit.name) { {{ fruit.name }} }

#31 sort

배열 정렬 (asc / desc)

Ascending:

@for (fruit of fruits | sort:'name':'asc'; track fruit.name) { {{ fruit.name }} }

Descending:

@for (fruit of fruits | sort:'name':'desc'; track fruit.name) { {{ fruit.name }} }
