Custom Directives Demo

15개의 실전 커스텀 디렉티브 예제

1. Highlight Directive

마우스를 올리면 배경색이 변경됩니다.

노란색 하이라이트

파란색 하이라이트

초록색 하이라이트

2. Underline Directive

마우스를 올리면 밑줄이 나타납니다.

빨간 밑줄 효과

파란 밑줄 효과

기본 밑줄 효과 (검정)

3. Tooltip Directive

마우스를 올리면 툴팁이 표시됩니다.

4. Click Outside Directive

드롭다운 외부를 클릭하면 닫힙니다.

@if (dropdownOpen()) {
  • 메뉴 1
  • 메뉴 2
  • 메뉴 3
}

5. Debounce Click Directive

빠르게 여러 번 클릭해도 300ms 내 마지막 클릭만 처리됩니다.

클릭 횟수: {{ debounceCount() }}

6. Lazy Load Image Directive

이미지가 화면에 보일 때 로드됩니다.

7. Auto Focus Directive

페이지 로드 시 자동으로 포커스됩니다.

8. Copy to Clipboard Directive

클릭하면 텍스트가 클립보드에 복사됩니다.

{{ copyText() }} @if (copyStatus()) { {{ copyStatus() }} }

9. Draggable Directive

요소를 드래그하여 이동할 수 있습니다.

드래그하세요!

10. Infinite Scroll Directive

스크롤이 끝에 도달하면 자동으로 더 로드합니다.

@for (item of scrollItems(); track item) {
{{ item }}
}

11. Has Permission Directive

사용자 권한에 따라 요소를 표시/숨깁니다.

read 권한 필요 콘텐츠
admin 권한 필요 콘텐츠

12. Character Counter Directive

입력 시 글자 수를 실시간으로 표시합니다.

13. Disable Button Directive

조건에 따라 버튼을 비활성화합니다.

14. Input Trim Directive

포커스를 벗어나면 앞뒤 공백이 자동 제거됩니다.

값: "{{ trimValue() }}"

15. Confirm Action Directive

클릭 시 확인 다이얼로그를 표시합니다.

@if (confirmMessage()) { {{ confirmMessage() }} }
